================================================================================
WHY SETUP IS REQUIRED
================================================================================

TAR Issue:
   TAR archives don't preserve the executable permission bit.
   All files come out as -rw-r--r-- (not executable).
   CGI scripts MUST be -rwxr-xr-x (executable).

Without SETUP.sh:
   ❌ CGI returns 404 (scripts not executable)
   ❌ DELETE may return 404 (test files missing)
   ❌ Tests fail with 60-70% pass rate

After SETUP.sh:
   ✅ CGI works (chmod +x applied to all scripts)
   ✅ DELETE works (all test files created)
   ✅ Tests pass with 85-95% pass rate

TROUBLESHOOTING
================================================================================

Problem: CGI returns 404
Fix: chmod +x cgi-bin/*.py cgi-bin/*.php

Problem: DELETE returns 404
Fix: echo "protected content" > www/delete_zone/protected.txt

Problem: Port 8080 in use
Fix: killall webserv

Problem: Tests failing
Fix: ./SETUP.sh
